O comando LG Magic Remote utiliza RF para comunicar com a televisão e, pelo menos a LG CX (ver que WebOS é que tem isto) existe a funcionalidade de
Universal Remote Controller.

Quando ligado, o setup pergunta que tipo de aparelho é e existe o Set-top box. Se ativado isto, o comando envia IR para a box.
Isto tem a vantagem de, no menu das apps, o icon "TV" salta para a box imediatamente.

Não arranjei forma de comunicar via Bleutooth com a box, mas se retirar a configuração que fiz do Universal Remote Controller, os controlos
vão via HDMI-CEC e resolveu o meu problema (porque a box está dentro de um móvel fechado).
